Lftp的使用

Lftp的安装
# Installing lftp
sudo yum -y install lftp
  1. Lftp的基本使用
先获取目标ftp地址
[train@localhost ~]$ lftp            #进入lftp
lftp :~> exit                        #退出lftp
[train@localhost ~]$ lftp
lftp :~> open ftp.ncbi.nlm.nih.gov   #进入ftp地址
lftp ftp.ncbi.nlm.nih.gov:~> ls
dr-xr-xr-x   4 ftp      anonymous     4096 Jul 17 02:48 1000genomes
-r--r--r--   1 ftp      anonymous 10738466816 Oct 17  2019 10GB
-r--r--r--   1 ftp      anonymous 1074790400 Oct 17  2019 1GB
-r--r--r--   1 ftp      anonymous 53688139776 Oct 17  2019 50GB
-r--r--r--   1 ftp      anonymous 5369757696 Oct 17  2019 5GB
最后使用get + 绝对路径进行下载
数据的下载
[train@localhost]$ lftp -e "get /blast/db/FASTA/nr.gz; exit" https://ftp.ncbi.nlm.nih.gov      #快捷下载方法，无需进入lftp
#还有-c -n等下载参数的调配

cd ok, cwd=/                                      `/blast/db/FASTA/nr.gz' at 52363264 (0%) 2.67M/s eta:14h [Receiving data]
数据的上传
lftp -e "put xxxx/xxxx/xxxx; exit" [url]   #url需要lft地址

第一个生信软件sratoolkit的安装

但是源为本地源已经下载好了,该软件为NCBI专用下载器

$ ls /opt                  
google

$ sudo mkdir /opt/biosoft /opt/sysoft       创建两个文件夹分别用于安装两类软件 生信专用软件和其他类别

$ ls /opt/
biosoft  google  sysoft

$ sudo chmod 1777 /opt/biosoft /opt/sysoft

$ ls -l /opt/
total 0
drwxrwxrwt 2 root root  6 Jul 18 09:40 biosoft
drwxr-xr-x 3 root root 20 Jul 17 17:11 google
drwxrwxrwt 2 root root  6 Jul 18 09:40 sysoft

$ tar -z -x -f /home/train/software/sratoolkit.current-centos_linux64.tar.gz -C /opt/biosoft/      #已经适配了centos8 所以可以直接解压直接使用即可 

$ ls /opt/biosoft/sratoolkit.2.11.0-centos_linux64/
bin  CHANGES  example  README-blastn  README.md  README-vdb-config  schema；

$ PATH=$PATH:/opt/biosoft/sratoolkit.2.11.0-centos_linux64/bin/    #添加环境变量，但仅为单次


$ vdb-config --interavtive    #需要配置软件下载位置

.bashrc的使用

针对以上的只能存在一次的问题，通过修改文件，每次登陆自动运行，使其成为永久生效的。

$ echo 'PATH=$PATH:/opt/biosoft/sratoolkit.2.11.0-centos_linux64/bin/  #sratoolkit' >> ~/.bashrc     #用字符与重新导向快捷添加

$ source ~/.bashrc #立即生效

该软件为ncbi专用下载软件
