接昨天在安装了GenomeScope2.0 后，在编译安装过程中，需要R等一系列环境，最终直接解压完编译后的压缩包，直接进行使用。

GS进行基因评估

jellyfish的安装

还需要进行依赖软件的安装

#wget https://github.com/gmarcais/Jellyfish/releases/download/v2.3.0/jellyfish-2.3.0.tar.gz -P ~/software/
$ tar zxf ~/software/jellyfish-2.3.0.tar.gz
$ cd jellyfish-2.3.0/
$ ./configure --prefix=/opt/biosoft/jellyfish-2.3.0
$ make -j 8 && make install
$ cd .. && rm -rf jellyfish-2.3.0
$ echo 'PATH=/opt/biosoft/jellyfish-2.3.0/bin:$PATH' >> ~/.bashrc
$ source  ~/.bashrc
使用genomescope 2.0进行基因组评估
$ mkdir -p ~/04.genome_assembling/genomescope2.0
$ cd ~/04.genome_assembling/genomescope2.0
$ ln -s /home/train/03.sequencing_data_quality_control/FastUniq/illumina.?.fastq ./
使用jellyfish进行k-mer分析
$ jellyfish count -C -m 21 -s 100000000 -t 4 -o mer_counts.jf *.fastq   #对K-mer进行计数

$ jellyfish dump mer_counts.jf > mer_counts.txt

$ jellyfish histo -t 4 mer_counts.jf > mer_counts.histo
用perl进行统计
$ perl -e '<>; <>; <>; <>; <>; while ($x = <>) { $x =~ s/.* //; $total += $x; } print "$total\n" ;' mer_counts.histo        #涉及到语法 针对特定文件 不进行解释
7126260
使用genomescope 2.0进行k-mer作图和基因组评估
$ /opt/biosoft/genomescope2.0-1.0.0/genomescope.R -i mer_counts.histo -o genomescope -k 21 -p 1 > genomescope.out eog genomescope/linear_plot.png

软件idba的安装

依旧是从本地安装 组装软件

$ ls ~/software/idba-1.1.3.tar.gz 
/home/train/software/idba-1.1.3.tar.gz
$ tar zxf /home/train/software/idba-1.1.3.tar.gz  -C /opt/biosoft/
$ ./configure --prefix=/opt/biosoft/idba-1.1.3
$ make install
$ echo 'PATH=$PATH:/opt/biosoft/idba-1.1.3/bin/' >> ~/.bashrc
$ source ~/.bashrc
使用 IDBA 进行基因组组装

二代基因组测序常用的组装方法
$ mkdir -p /home/train/04.genome_assembling/IDBA
$ cd /home/train/04.genome_assembling/IDBA
$ fq2fa –filter –merge ~/03.sequencing_data_quality_control/Trimmomatic/illumina.1.fastq ~/03.sequencing_data_quality_control/Trimmomatic/illumina.2.fastq illumina.fasta #准备输入文件

$ idba_ud -r illumina.fasta --mink 20 --maxk 100 --step 20
$ ls ./out/contig.fa

$ grep '>' out/contig.fa | les       # 列出输出文件中含 '>'标志的行


$ grep '>' out/contig.fa | wc -l     # 对含 '>' 的行统计数量
 
$ perl -e 'while (<>) { $total += length(<>) -1 ; } print "$total\n";' out/contig.fa        #统计总序列长度
