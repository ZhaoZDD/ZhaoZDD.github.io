生信分析学习Day1

发表于2022-07-17|更新于2022-07-17|生信
|字数总计:899|阅读时长:3分钟|阅读量:

文章内是对代码的整理，源代码见源代码

基础指令

一些使用Linux系统时常用的一些命令与功能

$ echo x （输出一些字符）示例如下
[train@localhost ~]$ echo 123
123
[train@localhost ~]$ x=123
[train@localhost ~]$ echo $x
123

$ pwd （显示当前位置）
$ cd  （进入文件夹）
$ ^C  （按键效果为ctrl+C 终止当前输入）
$ ~   （home）
$ /bin/ls   (输出当前目录)
$ /bin/cat  (打开路径文件)
$ /bin/less (打开路径文件) 和cat相比更建议使用less 退出文件Q键
一些常用快捷键

可以用于高效的进行操作

  • 复制 ctrl+shift+c
  • 粘贴 ctrl+shift+v or 选中区域后点击滚轮
  • 自动补齐 Tab键盘 #仅有一个时自动选择，当有两个以上首字母相同的会跳出，继续输入下一字母进行选择
PATH 环境变量

为了方便使用指令，不再使用绝对路径

$ PATH=$PATH:/xxx/xxx/.... (将新安装的软件包的指令添加到环境变量)
vim文本编辑器

一些基本操作，后续逐渐补充

正常非编辑模式，快速跳转指定行，可以ctrl+f和b进行前后翻页。
字母a进入编辑模式 再按ESC键退出编辑模式
保存输入指令 :wq! （w为保存，q为退出，!为强制退出）

vim官网www.vim.org可以了解更多功能

给用户账号root权限
$ vim /etc/sudoers （在第一百行添加）
root    ALL=(ALL)       ALL
train    ALL=(ALL)       ALL
更改系统安全系统
$ vim /etc/selinux/config 
SELINUX=disabled
配置文件修改

修改内存剩余多少时，使用虚拟内存

$ echo vm.swappiness=0 >> /etc/sysctl.conf 
$ cat /etc/sysctl.conf 查看结果
$ sysctl -p
$ sysctl vm.swappiness

示例如下
[root@localhost train]# sysctl vm.swappiness
vm.swappiness = 30
[root@localhost train]# echo vm.swappiness=0 >> /etc/sysctl.conf 
[root@localhost train]# cat /etc/sysctl.conf 
# sysctl settings are defined through files in
# /usr/lib/sysctl.d/, /run/sysctl.d/, and /etc/sysctl.d/.
# Vendors settings live in /usr/lib/sysctl.d/.
# To override a whole file, create a new file with the same in
# /etc/sysctl.d/ and put new settings there. To override
# only specific settings, add a file with a lexically later
# name in /etc/sysctl.d/ and put new settings there.
# For more information, see sysctl.conf(5) and sysctl.d(5).
vm.swappiness=0
[root@localhost train]# sysctl -p
vm.swappiness = 0
[root@localhost train]# sysctl vm.swappiness
vm.swappiness = 0
开放端口
$ systemctl restart firewalld.service （启动防火墙）

[root@localhost train]# firewall-cmd --add-port=80/tcp --permanent （添加端口）
You're performing an operation over default zone ('public'),
but your connections/interfaces are in zone 'libvirt' (see --get-active-zones)
You most likely need to use --zone=libvirt option.
--permanent表示永久生效
$ firewall-cmd --list-ports
80/tcp
查看当前IP地址
$ ifconfig （从报告中找） 本地地址为localhost
$ vim /var/www/html/index.html  （创建一个新的HTML网页）但是没有js和css文件 仅文本
允许访问目录
$ vim /etc/httpd/conf/httpd.conf 

$ alias /train "/home/train"
<Directory "/home/train">
    Options Indexes FollowSymLinks
    indexOptions FancyIndexing NameWidth=128 HTMLTable VersionSort FoldersFirst Charset=UTF-8
    AllowOverride None
    Order allow,deny
    Allow from all        （允许所有人访问#deny from 192.0.0.0）
    Require all granted
</Directory>
ESC退出编辑模式   :wq! 保存退出

$ systemctl restart httpd.service  (重启服务)

$ usermod -a -G train apache       #权限设置，给访客的
$ chmod 750 /home/train            #权限设置
$ systemctl restart httpd.service  #重启服务
在浏览器中输入 localhost/train 或者 对应的IP地址即可
挂载和卸载硬盘
ntfs-3g /dev/sda3 /mnt   (挂载)
umount /home            （卸载）
SH脚本使用

可以用于大量数据文件时的批量处理

#!/bin/bash

for x in `ls /`       #循环设置 /后为想要输出的文件路径
do
    echo "wc -l $x"   #在x变量前添加批处理指令
done
