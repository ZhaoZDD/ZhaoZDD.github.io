今天接触到了Obsidian这个学习笔记软件，这个软件可以和hexo有效的结合搭建更好的文章库，这里记录一些使用过程中学习到的东西

模板的使用

可以更方便快捷的导入Markdown的开头语法，除此之外也使用占位符实时更新当前的时间，省去了很多不必要的麻烦。目前遇到的缺点是不可以随便选择一个文件作为模板，必须要在建库的目录下进行选择，我的解决方法是建立一个模板文件夹，后续针对不同的情况进行导入和组合，应该有很多有意思的玩法

hexo 语法

#categories 可以为你的分类指定色彩，默认色 #333。

categories:
笔记: dimgray

#标语动画
首页的垂直交错排列文字效果。默认开启。
您可以访问 https://yunyoujun.cn 查看效果。（每次刷新，随机字体大小）

enable: 是否开启
title: 设置文字内容
border: 是否开启标语中字符的左右边框
cloud: 在首页下方显示流动的云
    enable: 是否开启
    color: 自定义色彩
go_down: 向下箭头按钮（点击翻页）

banner:
enable: true
title: 云游君的小站
src: /js/ui/banner.js
border: true
cloud:
enable: true
color: “white”
go_down:
enable: true
icon: icon-arrow-down-s-line
